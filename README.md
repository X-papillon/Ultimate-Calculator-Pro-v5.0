## ✨ Overview

Ultimate Calculator Pro is an all-in-one calculator extension that combines:

* Scientific Calculator
* Age Calculator
* Length Converter
* Weight Converter
* Speed Calculator
* Battery Life Calculator
* Wave Calculator

All inside one compact 450px Neon Dark popup.

---

## 🔥 Features

### 🧠 Scientific Calculator
* Powered by math.js (CSP safe)
* Supports:
  ```
  1+1
  2^3
  sqrt(25)
  sin(90)
  cos(45)
  tan(60)
  log10(100)
  pi
  e
  ```
* Clean button layout
* Fast evaluation
* No eval()
* Manifest V3 compliant

---

### 🎂 Age Calculator
* Calculate age from birth date
* Instant result in years

---

### 📏 Length Converter
* Meters
* Kilometers
* Centimeters
* Accurate real-time conversion

---

### ⚖ Weight Converter
* Kilograms
* Grams
* Pounds
* Fast unit switching

---

### 🚗 Speed Calculator
Formula:
```
Speed = Distance / Time
```
Result in km/h

---

### 🔋 Battery Life Calculator
Formula:
```
Battery Life (hours) = Capacity (Wh) / Power Consumption (W)
```
Displays runtime in hours

---

### 🌊 Wave Calculator
Formula:
```
Wave Speed = Frequency × Wavelength
```
Result in m/s

## 📦 Installation

1. Download the project
2. Open Chrome
3. Go to:
   ```
   chrome://extensions
   ```
4. Enable Developer Mode
5. Click "Load unpacked"
6. Select the extension folder
